## ADDED Requirements

### Requirement: Llista de sessions amb tema visual
El sistema SHALL mostrar la llista de sessions disponibles amb el tema de cinema fosc aplicat.

#### Scenario: Visualització de llista de sessions
- **WHEN** l'administrador accedeix a la pàgina de gestió de sessions
- **THEN** es mostren les sessions amb el tema visual aplicat

### Requirement: Dashboard de sessió
El sistema SHALL mostrar un dashboard visual per a cada sessió amb estat i estadístiques.

#### Scenario: Accés al dashboard
- **WHEN** l'administrador fa clic a Veure dashboard
- **THEN** es mostra el dashboard de la sessió

### Requirement: Formulari de creació de sessió
El sistema SHALL mostrar un formulari visual per crear noves sessions.

#### Scenario: Crear nova sessió
- **WHEN** l'administrador fa clic a Afegir sessió
- **THEN** es mostra el formulari de creació

### Requirement: Formulari d'edició de sessió
El sistema SHALL mostrar un formulari visual per editar sessions existents.

#### Scenario: Editar sessió
- **WHEN** l'administrador fa clic a Editar sessió
- **THEN** es mostra el formulari amb les dades

### Requirement: Botons d'estat de sessió
El sistema SHALL mostrar botons visuals per iniciar, editar o eliminar sessions.

#### Scenario: Gestió de sessió
- **WHEN** l'administrador gestiona una sessió
- **THEN** es mostren botons visuals per a cada acció
