## ADDED Requirements

### Requirement: Component AdminButton
El sistema SHALL disposar d'un component AdminButton reutilitzable per als botons d'acció.

#### Scenario: Ús de AdminButton
- **WHEN** es renderitza el component AdminButton
- **THEN** es mostra el botó amb l'estil del tema de cinema

### Requirement: Component AdminCard
El sistema SHALL disposar d'un component AdminCard per contenir contingut.

#### Scenario: Ús de AdminCard
- **WHEN** es renderitza el component AdminCard
- **THEN** es mostra la targeta amb l'estil del tema

### Requirement: Component AdminTable
El sistema SHALL disposar d'un component AdminTable per mostrar dades en taula.

#### Scenario: Ús de AdminTable
- **WHEN** es renderitza el component AdminTable
- **THEN** es mostra la taula amb estil visual

### Requirement: Component AdminForm
El sistema SHALL disposar d'un component AdminForm per a formularis.

#### Scenario: Ús de AdminForm
- **WHEN** es renderitza el component AdminForm
- **THEN** es mostra el formulari amb l'estil visual

### Requirement: Component AdminInput

#### Scenario: Ús de AdminInput

### Requirement: Component AdminSelect

#### Scenario: Ús de AdminSelect

- **THEN** es mostra el camp de selecció amb estil
