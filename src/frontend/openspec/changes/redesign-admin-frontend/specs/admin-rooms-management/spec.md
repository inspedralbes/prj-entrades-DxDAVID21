## ADDED Requirements

### Requirement: Llista de sales amb tema visual
El sistema SHALL mostrar la llista de sales de cinema amb el tema de cinema fosc aplicat.

#### Scenario: Visualització de llista de sales
- **WHEN** l'administrador accedeix a la pàgina de gestió de sales
- **THEN** es mostren les sales amb el tema visual

### Requirement: Formulari de creació de sala
El sistema SHALL mostrar un formulari visual per crear noves sales.

#### Scenario: Crear nova sala
- **WHEN** l'administrador fa clic a Afegir sala
- **THEN** es mostra el formulari de creació

### Requirement: Formulari d'edició de sala
El sistema SHALL mostrar un formulari visual per editar sales existents.

#### Scenario: Editar sala
- **WHEN** l'administrador fa clic a Editar
- **THEN** es mostra el formulari amb les dades

### Requirement: Capacitat de sales
El sistema SHALL mostrar la capacitat de cada sala de forma visual.

#### Scenario: Visualitzar capacitat
- **WHEN** es mostra la llista de sales
- **THEN** es mostra la capacitat de cada sala
